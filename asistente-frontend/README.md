# Asistente Frontend

Frontend web accesible para personas mayores que permite:

- Subir un archivo PDF
- Hacer preguntas en lenguaje natural
- Leer las respuestas en voz alta

## Instalación

```bash
npm install
npm run dev
```

## Producción

Usar `Vercel` para subir este frontend o correr `npm run build` y servirlo desde un backend.
